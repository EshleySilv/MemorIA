# 📚 MemorIA

Sistema inteligente de flashcards com Inteligência Artificial para apoio aos estudos.

## 🚀 Funcionalidades

- Criação de matérias (decks)
- Criação e gerenciamento de flashcards
- Sessão de estudo com cards individuais
- Sistema de avaliação (Fácil, Médio, Difícil, Errei)
- Cálculo de próxima revisão (repetição espaçada)
- Filtro automático de cards para revisar no dia

## 🧠 Objetivo

Auxiliar estudantes no processo de aprendizagem utilizando técnicas como:

- Active Recall
- Spaced Repetition
- Aprendizagem adaptativa

## 🛠 Tecnologias

- Python (Flask)
- SQLite
- HTML, CSS, JavaScript
